# Онлайн Суд - Многопользовательская судебная игра

Полноценная браузерная многопользовательская игра с real-time взаимодействием через WebSocket.

## 🎮 Особенности

### 🔐 Система входа
- **Вход как гость** - автоматическая генерация имени `Гость#12345678`
- **Discord OAuth** - вход через Discord (опционально)

### 👥 Система друзей
- Добавление друзей по имени
- Онлайн статус друзей
- Отображение кто в лобби
- Приглашение друзей в игру

### 🎯 Игровой процесс

#### 1. Создание лобби
- Хост создаёт лобби с уникальным ID
- Приглашает друзей в лобби
- Минимум 3 игрока для старта

#### 2. Распределение ролей
- **Судья** - ведёт процесс, выносит вердикт
- **Прокурор** - обвиняет подсудимого
- **Адвокат** - защищает подсудимого
- **Истец** - пострадавшая сторона

#### 3. Судебное слушание
- Изучение материалов дела
- Рассмотрение статей обвинения
- Обсуждение в Discord (голосовой чат)
- Ведение заметок по делу

#### 4. Вынесение вердикта
- Судья выносит решение
- Назначение наказания при обвинительном приговоре
- Завершение дела

### 📝 Дополнительные функции
- **Заметки** - записывайте важную информацию по делу
- **Real-time обновления** - все изменения синхронизируются в реальном времени
- **Система приглашений** - push-уведомления о приглашениях

### 🎨 Дизайн
- **Чёрно-серая гамма** с градиентами
- Минималистичный интерфейс
- Glassmorphism эффекты
- Адаптивная верстка

## 🚀 Запуск проекта

### Локально

```bash
# Установка зависимостей
npm install

# Сборка фронтенда
npm run build

# Запуск сервера
npm start
```

Сервер будет доступен на `http://localhost:3000`

### Разработка

Для разработки запустите два терминала:

**Терминал 1 - Frontend (Vite)**
```bash
npm run dev
```

**Терминал 2 - Backend (Node.js)**
```bash
npm run server
```

Frontend: `http://localhost:5173`
Backend: `http://localhost:3000`

## 🎭 Как играть

1. **Войдите в игру**
   - Нажмите "Войти как гость"
   - Вам будет присвоено имя `Гость#12345678`

2. **Добавьте друзей**
   - Откройте "Друзья"
   - Введите имя друга
   - Нажмите "+"

3. **Создайте лобби**
   - Нажмите "Создать лобби"
   - Получите ID лобби (например: `A1B2C3`)

4. **Пригласите друзей**
   - Нажмите "Пригласить друзей"
   - Выберите друга из списка
   - Друг получит уведомление

5. **Начните игру**
   - Дождитесь минимум 3 игроков
   - Хост нажимает "Начать игру"
   - Система генерирует случайное дело

6. **Выберите роли**
   - Каждый игрок выбирает свою роль
   - Судья, Прокурор, Адвокат, Истец

7. **Разберите дело**
   - Изучите материалы
   - Обсуждайте в Discord
   - Делайте заметки

8. **Вынесите вердикт**
   - Судья выносит решение
   - Назначает наказание
   - Дело завершается

## 🔧 API endpoints

### WebSocket события

**Клиент → Сервер:**
- `login:guest` - вход как гость
- `login:discord` - вход через Discord
- `friend:add` - добавить друга
- `friend:list` - получить список друзей
- `friend:remove` - удалить друга
- `lobby:create` - создать лобби
- `lobby:invite` - пригласить в лобби
- `lobby:join` - присоединиться к лобби
- `lobby:leave` - покинуть лобби
- `game:start` - начать игру
- `game:selectRole` - выбрать роль
- `game:verdict` - вынести вердикт

**Сервер → Клиент:**
- `friend:added` - друг добавлен
- `friend:removed` - друг удалён
- `lobby:updated` - лобби обновлено
- `lobby:invited` - приглашение в лобби
- `game:started` - игра началась
- `game:finished` - игра завершена

### HTTP endpoint
- `GET /api/health` - проверка работы сервера
