## 🎯 Как работает игра

### 1️⃣ Вход
```
Гость нажимает "Войти как гость"
  ↓
Автоматически создаётся имя: Гость#12345678
  ↓
Подключение к серверу через WebSocket
```

### 2️⃣ Добавление друзей
```
Открыть "Друзья"
  ↓
Ввести имя друга (например: Гость#87654321)
  ↓
Нажать "+"
  ↓
Друг появляется в списке
```

### 3️⃣ Создание лобби
```
Нажать "Создать лобби"
  ↓
Получить уникальный ID (например: A1B2C3)
  ↓
Стать хостом лобби
```

### 4️⃣ Приглашение в игру
```
Хост нажимает "Пригласить друзей"
  ↓
Выбирает друга из списка
  ↓
Друг получает push-уведомление
  ↓
Друг принимает/отклоняет
```

### 5️⃣ Начало игры
```
Минимум 3 игрока в лобби
  ↓
Хост нажимает "Начать игру"
  ↓
Сервер генерирует случайное дело
  ↓
Все игроки видят материалы дела
```

### 6️⃣ Выбор ролей
```
Каждый игрок выбирает роль:
- Судья (1 человек)
- Прокурор (1 человек) - обвиняет
- Адвокат (1 человек) - защищает
- Истец (1+ человек) - пострадавший
```

### 7️⃣ Разбор дела
```
Все читают материалы дела
  ↓
Обсуждают в Discord (голосовой чат)
  ↓
Прокурор обвиняет
  ↓
Адвокат защищает
  ↓
Истец даёт показания
```

### 8️⃣ Вердикт
```
Судья выносит решение:
- Виновен → Назначает наказание
- Не виновен → Дело закрывается
  ↓
Игра завершается
  ↓
Все возвращаются в лобби
```

## 🎨 Дизайн решения
