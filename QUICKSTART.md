# 🚀 Быстрый старт

## Для разработчика

### 1. Установка
```bash
npm install
```

### 2. Локальный запуск

**Вариант 1: Production режим**
```bash
npm run build
npm start
```
Откройте `http://localhost:3000`

**Вариант 2: Development режим**

Терминал 1 (Frontend):
```bash
npm run dev
```

Терминал 2 (Backend):
```bash
npm run server
```
Откройте `http://localhost:5173`

### 3. Deploy на Railway
```bash
git add .
git commit -m "Ready for deploy"
git push
```
Затем подключите репозиторий на railway.app

## Для игрока

### 1. Вход
1. Откройте сайт
2. Нажмите **"Войти как гость"**
3. Вам автоматически присвоится имя `Гость#12345678`

### 2. Добавить друзей
1. Нажмите кнопку **"Друзья"** в шапке
2. Попросите друга назвать его имя (например: `Гость#87654321`)
3. Введите имя друга в поле
4. Нажмите **"+"**
5. Друг появится в списке

### 3. Создать игру
1. На главном экране нажмите **"Создать лобби"**
2. Вы станете хостом лобби
3. Нажмите **"Пригласить друзей"**
4. Выберите друга и нажмите **"Пригласить"**
5. Друг получит уведомление и сможет присоединиться

### 4. Начать игру
1. Дождитесь минимум **3 игроков** в лобби
2. Хост нажимает **"Начать игру"**
3. Система автоматически создаст дело

### 5. Выбрать роль
Каждый игрок выбирает одну из ролей:
- **Судья** - выносит вердикт
- **Прокурор** - обвиняет подсудимого
- **Адвокат** - защищает подсудимого
- **Истец** - пострадавший

### 6. Разобрать дело
1. Прочитайте **материалы дела**
2. Изучите **статьи обвинения**
3. Откройте **Discord** и создайте голосовой канал
4. Обсудите дело всей командой
5. Используйте **"Заметки"** для записи важного

### 7. Вынести вердикт
1. **Судья** выбирает: "Виновен" или "Не виновен"
2. Если виновен - назначает наказание
3. Нажимает **"Огласить вердикт"**
4. Все видят результат

### 8. Следующее дело
1. Нажмите **"Завершить"**
2. Вернётесь в лобби
3. Можете начать новую игру

## 💡 Советы

### Для хоста
- Приглашайте друзей через кнопку в лобби
- Дождитесь минимум 3 игроков
- Начинайте игру когда все готовы

### Для прокурора
- Ваша задача - доказать вину
- Используйте статьи обвинения
- Обсуждайте материалы дела

### Для адвоката
- Ваша задача - защитить подсудимого
- Ищите несоответствия в обвинении
- Защищайте права обвиняемого

### Для истца
- Расскажите что произошло
- Дайте показания
- Помогите прокурору

### Для судьи
- Слушайте обе стороны
- Изучите материалы
- Вынесите справедливый вердикт

## 🎮 Discord интеграция

### Рекомендуемая схема
1. Создайте Discord сервер
2. Создайте голосовой канал "Судебное заседание"
3. Все игроки подключаются к каналу
4. Обсуждайте дело голосом
5. Используйте текстовый чат для доказательств

### Роли в Discord (опционально)
- Создайте роли: Судья, Прокурор, Адвокат, Истец
- Выдайте участникам соответствующие роли
- Это поможет организовать обсуждение

## ❓ FAQ

**Q: Сколько нужно игроков?**
A: Минимум 3, оптимально 4-6

**Q: Можно ли играть без Discord?**
A: Да, но общение голосом сильно улучшает игру

**Q: Как добавить друга?**
A: Узнайте его имя (Гость#12345678) и добавьте в списке друзей

**Q: Что если я хочу сменить роль?**
A: Попросите хоста перезапустить игру

**Q: Можно ли протестовать вердикт?**
A: Нет, решение судьи окончательное
