## 🔄 Как обновить существующий проект

### Локально
```bash
# 1. Получить обновления
git pull

# 2. Переустановить зависимости
npm install

# 3. Пересобрать проект
npm run build

# 4. Запустить
npm start
```

### На Railway
```bash
# 1. Закоммитить изменения
git add .
git commit -m "Update to fixed version"
git push

# 2. Railway автоматически передеплоит
# 3. Готово!
```

## 💡 Опциональная настройка Discord

Если хотите Discord OAuth:

```bash
# 1. Создайте .env файл
echo "VITE_DISCORD_CLIENT_ID=ваш_client_id" > .env

# 2. Перезапустите
npm run build
npm start
```

Подробнее в [DISCORD_SETUP.md](DISCORD_SETUP.md)
