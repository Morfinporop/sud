# 🎮 Настройка Discord OAuth

## Зачем нужен Discord OAuth?

- Вход под своим Discord аккаунтом
- Отображение вашего аватара
- Уникальный идентификатор вместо случайного ID
- Сохранение прогресса между сессиями

## 📋 Пошаговая инструкция

### 1. Создайте Discord приложение

1. Перейдите на [Discord Developer Portal](https://discord.com/developers/applications)
2. Войдите в свой Discord аккаунт
3. Нажмите **"New Application"**
4. Введите название приложения (например: "Онлайн Суд")
5. Нажмите **"Create"**

### 2. Настройте OAuth2

1. В левом меню выберите **"OAuth2"**
2. Найдите раздел **"Client ID"**
3. Скопируйте Client ID (длинная строка цифр)
4. Нажмите **"Add Redirect"** в разделе Redirects
5. Добавьте следующие URL:
   - `http://localhost:5173` (для разработки)
   - `http://localhost:3000` (для production локально)
   - `https://ваш-домен.railway.app` (для Railway)

### 3. Локальная разработка

Создайте файл `.env` в корне проекта:

```env
VITE_DISCORD_CLIENT_ID=ваш_client_id_здесь
```

**Пример:**
```env
VITE_DISCORD_CLIENT_ID=1234567890123456789
```

### 4. Railway Deploy

**Discord Client ID уже настроен!** (1500581551069462569)

Но если нужно изменить:
1. Откройте ваш проект на Railway
2. Перейдите в **Settings** → **Variables**
3. Добавьте/измените переменную:
   - **Key:** `VITE_DISCORD_CLIENT_ID`
   - **Value:** ваш Client ID
4. Сохраните и передеплойте проект

**Важно для Railway:**
В настройках Discord приложения добавьте Redirect URI:
```
https://ваш-проект.up.railway.app
```

### 5. Проверка

После настройки:
1. Перезапустите приложение
2. На экране входа появится кнопка **"Войти через Discord"**
3. Нажмите на неё
4. Разрешите доступ к вашему Discord аккаунту
5. Вы будете перенаправлены обратно в игру

## 🔒 Безопасность

### ✅ Что безопасно
- Client ID - это публичная информация
- Можно хранить в переменных окружения
- Можно коммитить в `.env.example`

### ❌ Что НЕ безопасно
- **Client Secret** - НЕ используется в этом проекте
- Если потребуется - храните только на сервере
- Никогда не коммитите в Git

## 📱 Пример использования

### Без Discord OAuth:
```
Вход как гость → Гость#12345678
```

### С Discord OAuth:
```
Вход через Discord → YourUsername#1234
(с вашим аватаром и постоянным ID)
```

## 🛠 Отладка

### Проблема: Кнопка Discord неактивна
**Решение:** Проверьте что `VITE_DISCORD_CLIENT_ID` установлен

### Проблема: "Invalid redirect_uri"
**Решение:** Добавьте текущий URL в Discord Redirects

### Проблема: "Invalid client"
**Решение:** Проверьте правильность Client ID

### Проблема: Бесконечная загрузка
**Решение:** 
1. Очистите кэш браузера
2. Очистите localStorage
3. Проверьте консоль на ошибки

| Аватар | ❌ | ✅ |
| Настоящий ник | ❌ | ✅ |
| Сохранение данных | Локально | Локально + Discord ID |
