# 🤖 Боты и присоединение к лобби

## ✅ Что добавлено

### 1. Присоединение по ID лобби
Теперь можно присоединиться к существующему лобби:
1. На главном экране нажмите **"Присоединиться"**
2. Введите ID лобби (например: `5HEMNB`)
3. Нажмите **"Присоединиться"**

### 2. Тестовые боты
Добавлена система ботов для тестирования:
- 🤖 Прокурор (роль: прокурор)
- 🤖 Адвокат (роль: адвокат)
- 🤖 Судья (роль: судья)
- 🤖 Истец (роль: истец)

### 3. Добавление ботов
В лобби хост может:
1. Нажать кнопку **"🤖 Бота"**
2. Бот автоматически добавится в лобби
3. Максимум 6 игроков (включая ботов)

## 🎮 Как использовать

### Создать лобби и добавить ботов
```
1. Нажмите "Создать лобби"
2. Получите ID (например: 5HEMNB)
3. Нажмите "🤖 Бота" 2-3 раза
4. Теперь у вас 3+ игроков
5. Нажмите "Начать игру"
```

### Присоединиться к лобби
```
1. Получите ID лобби от друга
2. Нажмите "Присоединиться"
3. Введите ID (например: 5HEMNB)
4. Нажмите "Присоединиться"
5. Вы в лобби!
```

## 🐛 Решение проблем

### Не могу присоединиться к лобби
**Причина:** ID лобби неправильный или лобби не существует
**Решение:**
- Проверьте ID (должен быть 6 символов)
- ID чувствителен к регистру
- Лобби могло быть удалено

### Не могу добавить бота
**Причина:** Лобби полное или боты закончились
**Решение:**
- Максимум 6 игроков
- Максимум 4 бота (по одному типу)

### Бот не появляется
**Причина:** Проблема с сервером
**Решение:**
- Обновите страницу
- Проверьте консоль на ошибки
- Перезапустите сервер
